## Compile & Run

### Requirements
Linux:
- Required packages: `cmake`, `make`, `gcc`, `libncurses-dev`
- Rust compiler must be installed [Rust installation](https://www.rust-lang.org/tools/install)

Windows:
- Install cmake and add it to $PATH
- Install MinGW and add it to $PATH
- Rust compiler must be installed [Rust installation](https://www.rust-lang.org/tools/install)

### Compile & Run

1. `cargo build`
2. `cargo run`
